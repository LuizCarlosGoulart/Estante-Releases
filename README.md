# Estante 📚

**Estante** é a sua biblioteca de documentos offline, projetada para oferecer uma experiência de leitura focada, organizada e eficiente.

---

## 🚀 Como Instalar (Direto)

Para instalar o **Estante** de forma simples e rápida:

1. Clique no botão verde **"<> Code"** no topo desta página.
2. Selecione a opção **"Download ZIP"**.
3. Após baixar, extraia o conteúdo do arquivo `.zip` em uma pasta de sua preferência.
4. Dentro da pasta extraída, execute o arquivo: `Estante-Setup.exe`.
5. Siga as instruções do instalador para concluir a instalação no seu computador.

> [!NOTE]
> O **Estante** será instalado permanentemente no seu sistema e um atalho será criado na sua Área de Trabalho.

---

## ✨ Destaques

- 📁 **Organização Intuitiva**: Gerencie sua biblioteca de PDFs com facilidade.
- 🔍 **Leitor Integrado**: Experiência de leitura fluida e rápida.
- 💾 **100% Offline**: Seus dados e documentos permanecem no seu computador.

---

## 🛠️ Requisitos do Sistema

- **Sistema Operacional**: Windows 10 ou 11 (64-bit).
- **Espaço em Disco**: ~200MB para a instalação.

---

## ❓ Solução de Problemas

### Alerta do Windows SmartScreen?
Como o software é distribuído de forma independente, o Windows pode exibir um alerta. 
- Clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.
