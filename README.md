# Estante 📚

**Estante** é a sua biblioteca de documentos offline, projetada para oferecer uma experiência de leitura focada, organizada e eficiente.

[![Release](https://img.shields.io/github/v/release/LuizCarlosGoulart/Estante-Releases?label=Vers%C3%A3o%20Est%C3%A1vel&color=blue)](https://github.com/LuizCarlosGoulart/Estante-Releases/releases)
[![Platform](https://img.shields.io/badge/Plataforma-Windows-blue)](https://github.com/LuizCarlosGoulart/Estante-Releases/releases)

---

## ✨ Destaques

- 📁 **Organização Intuitiva**: Gerencie sua biblioteca de PDFs com facilidade.
- 🔍 **Leitor Integrado**: Experiência de leitura fluida e rápida.
- 💾 **100% Offline**: Seus dados e documentos permanecem no seu computador.
- 🚀 **Performance**: Construído para ser leve e responsivo.

---

## 🚀 Como Instalar

Para começar a usar o **Estante**, siga os passos abaixo:

1. Acesse a seção de [**Releases (Lançamentos)**](https://github.com/LuizCarlosGoulart/Estante-Releases/releases/latest).
2. Baixe o arquivo mais recente: `Estante Setup 1.1.0.exe`.
3. Execute o instalador e siga as instruções na tela.
4. O **Estante** será instalado no seu computador e um atalho será criado na Área de Trabalho.

---

## 🛠️ Requisitos do Sistema

- **Sistema Operacional**: Windows 10 ou 11 (64-bit).
- **Espaço em Disco**: ~200MB para a instalação.
- **Memória**: 4GB RAM recomendado.

---

## ❓ Solução de Problemas

### O SmartScreen do Windows bloqueou a instalação?
Como o software ainda não possui uma assinatura digital (code signing), o Windows pode exibir um alerta. 
- Clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.

### Preciso de internet para usar?
Não! O **Estante** funciona de forma totalmente offline. A internet é necessária apenas para baixar o instalador inicial ou atualizações.
